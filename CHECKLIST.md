# 受け入れチェックリスト

以下の項目について動作を確認してください。

## 1. 基本動作
- [ ] `npm run dev` でエラーなく起動するか
- [ ] ブラウザでアクセスできるか (http://localhost:3001)
- [ ] レスポンシブ表示で崩れていないか (スマホサイズ推奨)

## 2. ホーム画面
- [ ] 「五行精霊と漢方図鑑」のタイトル表示
- [ ] 精霊3体（木：モク、火：カ、土：ド）が表示されている
- [ ] 「遊ぶ」「図鑑」ボタンがタップ可能

## 3. ミニゲーム
### 相生チェイン (/play/chain)
- [ ] スタートボタンで開始
- [ ] 制限時間のカウントダウン
- [ ] スワイプ（またはタップ）で属性アイコンをつなげる
- [ ] 正解（相生順）でスコア加算
- [ ] 終了後にリザルト画面が出る

### 相克ガード (/play/guard)
- [ ] 暴走している属性（中央）に対し、勝てる属性を選択肢から選ぶ
- [ ] 正解でスコア加算、不正解で減点
- [ ] 終了後にリザルト画面が出る

### 連想仕分け (/play/sort)
- [ ] キーワード（例：怒り、赤、甘味）が表示される
- [ ] 正しい属性（木・火・土）のエリアへドラッグ
- [ ] 終了後にリザルト画面が出る

## 4. 図鑑・育成
- [ ] ゲーム終了後、漢方カードを獲得できる
- [ ] ホーム画面に戻ると、獲得した分だけの変化があるか（裏側でステータス加算など）
- [ ] 図鑑 (/dex) で、獲得したカードがシルエットからカラーになっている
- [ ] カードをタップすると詳細（フレーバーテキスト、効果）が見られる
- [ ] リロードしてもデータ（獲得状況）が残っている (localStorage)

## 5. PWA / その他
- [ ] 日本語フォント (Noto Sans JP) が適用されている
- [ ] 課金要素が表示されていないこと
- [ ] エラーでアプリが落ちないこと
