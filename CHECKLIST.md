# 受け入れチェックリスト

以下の項目について動作を確認してください。

## 1. 基本動作
- [ ] `npm run dev` でエラーなく起動するか
- [ ] ブラウザでアクセスできるか (http://localhost:3001)
- [ ] レスポンシブ表示で崩れていないか (スマホサイズ推奨)

## 2. ホーム画面・導入
- [ ] 起動時に世界観を伝える「ストーリーモーダル」が表示されるか
- [ ] 「生薬図鑑」ボタンが表示されているか
- [ ] 各要素（精霊の元気など）が反映されているか

## 3. 修行と収集
- [ ] ミニゲーム終了後、報酬として「生薬」を獲得できるか
- [ ] リザルト画面で獲得した生薬のアイコンが正しく表示されるか

## 4. 図鑑と調合
- [ ] 図鑑画面で「基本生薬」と「漢方調合」のタブが切り替えられるか
- [ ] 未習得のアイテムでもクリックしてレシピを確認できるか
- [ ] 「漢方調合」タブが初期状態でロック（アップグレード案内）されているか
- [ ] 「アップグレード」ボタンで調合機能が解放されるか
- [ ] 必要な生薬を消費して「調合」を実行し、漢方を生成できるか

## 5. 育成（ヒーリング）
- [ ] 調合した漢方を使って、精霊の「ごきげん」を回復できるか
- [ ] 属性一致ボーナス（1.5倍）が正しく計算されているか
- [ ] 回復アニメーションが表示され、ホームに戻ると元気が増えているか
