# 五行精霊と漢方図鑑 最終仕様書 (v1.0)

## 1. プロダクト概要
五行（木火土金水）の概念を、説明を読ませず「遊びながら自然に覚える」ことを目的としたスマートフォン向けWebアプリケーション（PWA）。

## 2. 技術スタック
- **Frontend**: Next.js (App Router), TypeScript
- **Styling**: Tailwind CSS (White Base Theme / Light Mode)
- **State Management**: Zustand
- **Persistence**: localStorage (Hydrate/Save)
- **Animations**: Framer Motion, Lucide React (Icons)
- **Hosting**: Vercel (Deployed via GitHub)

## 3. 主要機能
### A. ホーム画面
- **精霊カルーセル**: 初期は「木・火・土」の3体が解放。各精霊の気分（絶好調・ふつう・お疲れ）と元気度を表示。
- **精霊のお願い**: ランダムで発生するクエスト。対応する修行をクリアすると報酬ボーナス。
- **サポートキャラ案内**: ストーリー導入や新カード獲得時のポップアップ通知。
- **設定/ヘルプ**: 物語を再確認可能。
- **解放条件**:
  - コン（金）: 合計習得度 500 で自動解放
  - スイ（水）: 合計習得度 1000 で自動解放

### B. 修行（ミニゲーム）
1. **相生チェイン**: 相生(木→火→土→金→水)の順に属性を繋ぐ。ターゲット出現保証ロジック搭載。
2. **相克ガード**: 暴走する属性に対し、相克（勝てる属性）で抑え込む3択ゲーム。
3. **連想仕分け**: 季節、感情、色などのキーワードを正しい五行（木・火・土）にドラッグで仕分ける。

### C. 漢方図鑑
- 初期20枚のカードを収録（木×3, 火×4, 土×6, 金×2, 水×5 ※重複あり）。
- 獲得したカードは詳細（フレーバーテキスト、漢方解説）を閲覧可能。
- 新規獲得時はホーム画面でキャラクターがお知らせし、図鑑ボタンが点滅。

## 4. UI/UX デザイン方針
- **清潔感**: 白ベースの背景に、五行のテーマカラー（緑・赤・黄・灰・青）を差し色として使用。
- **視認性**: 日本語メイン、太字フォントによる快適な読み心地。
- **PWA**: ホーム画面に追加可能なメタデータとアイコン設定。

## 5. デプロイ手順
1. GitHubリポジトリ `akisuperprof-sketch/gogyou-training-APP` に接続。
2. Vercelのダッシュボードからプロジェクトを作成。
3. デフォルト設定でデプロイ（ポート設定などはVercel側で自動処理）。

## 6. 保存データ
- `gogyou-storage-v4`: localStorageに全ての進捗（精霊ステータス、図鑑収集、クリア履歴）を保存。
